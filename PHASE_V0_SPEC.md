# Phase 0 (V0.1.0) - 基础地图与单原点单终点

## 版本目标

实现最基础的风水罗盘功能：地图显示、单原点、单终点、连线测量。

## 功能清单

### 1. 地图界面

#### 1.1 GPS权限请求
- 打开应用时自动跳转到系统定位权限授予界面（允许/禁止选项）
- 若用户拒绝授予定位权限：
  - 正常显示地图和罗盘
  - 每次打开应用时提示"需要位置权限才能使用定位功能"
  - 其余功能不中断

#### 1.2 地图基础功能
- 地图类型切换：矢量图 ↔ 卫星图
- 缩放控制：通过屏幕右侧按键区加减号切换比例尺
- 进入软件后罗盘出现在当前GPS处并固定

#### 1.3 屏幕中心十字指示
- 屏幕中心出现十字指示当前屏幕中心位置
- 该十字指示仅在"地图"页面出现
- 十字在地图界面一直固定在地图中心

### 2. 罗盘显示

#### 2.1 罗盘UI
- 参考Flutter项目的UI设计
- 罗盘出现在当前GPS处并固定
- 罗盘尺寸：1000x1000像素（约3.8MB内存）

### 3. 原点与终点管理

#### 3.1 添加原点
**触发方式：** 点击小加号或屏幕中心十字指示

**行为：**
- 在当前位置新建原点
- 若此时不存在该项目终点：
  - 罗盘出现在原点处
  - 存储该原点
- 若此时存在该项目终点：
  - 产生原点终点连线
  - 罗盘出现在原点处
  - 存储该原点

#### 3.2 添加终点
**触发方式：** 点击小加号或屏幕中心十字指示

**行为：**
- 在当前位置新建终点
- 若此时不存在该项目原点：
  - 存储该终点
  - 地图显示无变化
- 若此时存在该项目原点：
  - 产生原点终点连线
  - 罗盘出现在原点处
  - 视角沿着连线移回该原点
  - 终点位置用半透明方框显示：
    - 24山位置
    - 点位名称
    - 距离和方位
  - **字体重要性：** 24山 > 点位名称 > 距离和方位

### 4. 连线信息显示

#### 4.1 点击连线
**触发方式：** 点击两点间连线

**显示信息：**
- 原点和终点的名称
- 原点和终点的经纬度
- 二者之间的方位角
- 24山
- 八卦
- 五行
- 直线距离

## 数据模型

### GeoPoint (简化版)
```swift
struct GeoPoint {
    let id: Int
    let name: String
    let latitude: Double
    let longitude: Double
    let pointType: PointType  // origin / destination
}

enum PointType: String {
    case origin
    case destination
}
```

### Connection (连线信息)
```swift
struct Connection {
    let origin: GeoPoint
    let destination: GeoPoint
    let distance: Double        // 米
    let bearing: Double         // 0-360°
    let shanName: String        // 24山名称
    let baguaName: String       // 八卦名称
    let wuxingName: String      // 五行名称
}
```

## 技术实现要点

### 1. 地图SDK初始化
- 使用高德地图SDK（AMapFoundationKit + AMap3DMap）
- 隐私合规配置（见ARCHITECTURE.md 12.3节）

### 2. 罗盘渲染
- 使用GroundOverlay在地图上绘制罗盘图片
- 罗盘图片预渲染（1000x1000 ARGB）
- 罗盘方向固定指向正北（不跟随设备传感器）

### 3. 连线绘制
- 使用Polyline绘制原点到终点的连线
- 线宽：12像素
- 颜色：使用ARCHITECTURE.md第8节定义的第一种颜色

### 4. 方位角计算
- 使用Rhumb Line算法计算方位角（见ARCHITECTURE.md 3.1节）
- 使用Vincenty算法计算距离（见ARCHITECTURE.md 3.2节）
- 方位角转24山（见ARCHITECTURE.md 3.3节）

## 已知限制

- 仅支持单原点单终点
- 不支持案例管理
- 不支持罗盘锁定/解锁模式
- 不支持定位按钮
- 罗盘固定在原点位置，不可移动
